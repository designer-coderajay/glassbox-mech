Quickstart

One function call. Full circuit report.

No PhD required. Install and run in under 5 minutes.

$ pip install glassbox-mech-interp
from transformer_lens import HookedTransformer from glassbox import GlassboxV2 model = HookedTransformer.from_pretrained("gpt2") gb = GlassboxV2(model) result = gb.analyze( prompt = "When Mary and John went to the store, John gave a drink to", correct = " Mary", incorrect = " John", ) print(result["circuit"]) # [(9, 9), (9, 6), (10, 0)] <- (layer, head) tuples, sorted by attribution print(result["faithfulness"]) # { 'sufficiency': 1.00, 'comprehensiveness': 0.47, 'f1': 0.64, # 'category': 'backup_mechanisms', 'suff_is_approx': True }

Capabilities

Everything in one toolkit.

Built on TransformerLens. Works on any TransformerLens-compatible model.

⚡
Attribution Patching
Per-head causal importance via Jacobian x delta-z. 3 forward passes. Fast enough for production use.
Nanda 2023
🔬
Minimum Faithful Circuit
Automated circuit discovery via asymmetric greedy algorithm. Forward selection (approximate) + backward pruning (exact causal). O(3 + 2p) passes.
Novel algorithm
📐
FCAS — Circuit Alignment
Functional Circuit Alignment Score: compare circuit structure across models using relative depth matching with a bootstrap null distribution and z-score.
Novel metric
🔮
Logit Lens
Track how any token's probability evolves through the residual stream layer by layer. Includes per-head direct effects at each layer.
nostalgebraist 2020
📊
Bootstrap Confidence Intervals
95% CI on sufficiency, comprehensiveness, and F1 via nonparametric bootstrap (n=500). Statistically grounded faithfulness claims across N prompts.
500 resamples
✅
Honest Approximations
Every approximation is explicitly flagged. suff_is_approx: True in every output. Comprehensiveness is always exact. Nothing hidden.
Production-grade
🔗
Head Composition Scores
Q, K, V composition scores between any two attention heads. Full implementation of the Elhage et al. (2021) virtual weights framework.
Elhage 2021
🌐
No-Install Web Demo
HuggingFace Space with attribution heatmap, logit lens, and attention visualisation. Paste a prompt, get a circuit report in seconds.
Live on HF Spaces
🧬
SAE Feature Attribution
Decompose circuit heads into sparse SAE features with Neuronpedia links. Optional sae-lens dependency. Currently supports GPT-2 small.

Enforcement starts August 2026.

Article 13 of the EU AI Act requires high-risk AI systems — in finance, healthcare, HR, and legal — to provide meaningful, human-readable explanations of their decisions. Glassbox's structured output maps directly to these requirements.

  • Identifies which specific model components (layer, head) causally drove a prediction
  • Quantifies sufficiency: how much of the prediction the identified circuit explains
  • Quantifies comprehensiveness: causal necessity, not just correlation
  • Outputs structured JSON suitable for audit documentation and regulatory review
  • Bootstrap confidence intervals provide statistical grounding for faithfulness claims
  • Every approximation is explicitly disclosed — no misleading precision
