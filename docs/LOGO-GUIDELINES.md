## The Mark

The Glassbox AI mark is a **rounded square (glass box) containing a four-node diamond circuit**. The nodes represent the key attention heads identified in the peer-reviewed research (arXiv:2603.09988): primary output (top, largest), S-inhibition (left), name mover (right), input embedding (bottom). The dashed arc connecting left and right nodes references the S-inhibition mechanism. Every element maps directly to the science.

The frame gradient flows from fire orange (warm, human-facing output) → ice blue (cool, internal computation). This thermal metaphor runs through the entire design system.

## Logo Usage Rules

### ✓ Do
- Use the mark on dark backgrounds (`#050709` or equivalent near-black)
- Maintain minimum clear space of 1× mark height on all sides
- Use the full lockup (mark + wordmark) in nav bars and marketing
- Use the mark alone at 16px–64px for icons and favicons
- The gradient frame works best at 24px and above

### ✗ Do not
- Recolor the mark (fire→ice gradient is identity-critical)
- Place on white/light backgrounds without the dark container
- Stretch or distort the square aspect ratio
- Add drop shadows or overlays that obscure the circuit traces
- Use a different font for the wordmark

## Competitive Inspiration

The mark draws from 50+ competitor analyses across AI, developer tools, and enterprise software:

**Geometric precision**: OpenAI (abstract bloom), Anthropic (geometric A), Stripe (wordmark), Linear (asterisk), Vercel (triangle) — all prove that the strongest marks reduce to one clean geometric idea.

**Technical vocabulary**: NVIDIA, Palantir, Scale AI, Databricks — geometric forms that suggest precision engineering rather than consumer warmth.

**Interpretability/observability**: Arize AI, Fiddler, Arthur AI, WhyLabs, Evidently — the niche that Glassbox owns, none of which have a mark as conceptually precise as the circuit-in-glass-box.

**The insight**: Every major AI brand owns *one shape*. Glassbox owns the glass box with the visible neural circuit — a mark that only this company can legitimately use, because it references the actual science.
